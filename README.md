# Управление памятью и фундаментальные структуры данных в C++: массивы, указатели, строки
## Описание проекта
Этот проект представляет собой консольное приложение на языке C++, реализующее набор интерактивных утилит для различных задач: анализ успеваемости группы студентов, умный калькулятор с историей операций, детектор палиндромов и анаграмм, работа с матрицами и система управления библиотекой книг. Программа запускается из main() с меню выбора модулей (1-5), где каждый модуль предлагает подменю для повторных операций. Код написан в учебных целях, демонстрируя динамическую память, структуры, строки и алгоритмы.

## Модули программы
1. Анализ успеваемости (GroupPerformanceAnalyzer): Генерирует оценки 15 студентов (2-5 баллов), вычисляет средний балл, число двоечников (<3), процент отличников (>5, но баллы до 5), наиболее повторяющуюся оценку.

2. Умный калькулятор (SmartCalculatorWithTransactionHistory): Арифметика (+, -, *, /, %), обмен значений, поиск min/max в массиве, история до 5 операций.

3. Детектор палиндромов/анаграмм (PalindromeAnagramDetector): Проверка палиндрома (игнор регистр/пробелы), анаграмм, реверс строки, подсчет слов, поиск longest word, очистка неалфавитных символов.

4. Матрицы (TransposingMatrixArbitrarySize): Заполнение случайными числами (1-10), транспонирование, суммирование элементов по позициям для произвольных размеров.

5. Библиотека книг (BookLibraryManagementSystem): Добавление книг (название, автор, год; дубликаты по году не добавляются), поиск по названию/автору, удаление, вывод списка.

## Особенности реализации
Код использует динамические массивы (new/delete), ручное управление памятью, циклы меню с обработкой ошибок ввода (std::cin.fail(), ignore()).
